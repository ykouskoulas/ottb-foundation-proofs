Define atan2, a two-argument atan function whose return angle acconts for the sign of each argument. It has a branch cut at PI and a range in (-PI,PI].

Definition pre_atan2 (dy dx : R) : {θ : R | -PI < θ ≤ PI ∧
                                        dy = sqrt (dy² + dx²) × sin θ ∧
                                        dx = sqrt (dy² + dx²) × cos θ}.

Definition atan2 y x := let (v, _) := pre_atan2 y x in v.

Lemma atan2_val : ∀ dy dx θ, - PI < θ ≤ PI → (dy ≠ 0 ∨ dx ≠ 0) →
